# #17 구현 계획 - 상세 패널 및 컨텍스트 메뉴 도입

## 수정 목적
- 파일 단건 메타 확인을 빠르게 하고, 우클릭 기반의 데스크톱형 액션 진입점을 제공한다.

## 수정할 부분
1. `FileTable`
   - 행 우클릭 이벤트 노출(`onContextMenu`)
2. `FileBrowser`
   - 컨텍스트 메뉴 UI 추가(열기/선택/정보)
   - 우측 상세 패널(이름/타입/크기/수정일/소유자) 추가
3. 선택 상태 연계
   - 우클릭 시 해당 항목 포커싱 및 상세 패널 동기화

## 테스트 계획
- frontend: `npx vitest run`
- frontend: `npm run build`
- 수동: 우클릭 메뉴 표시/닫힘, 상세패널 값 동기화 확인
